# C# Practice Projects ‚Äì Beginner Console Apps

This repository contains a collection of beginner-to-intermediate level **C# console applications**.  
Each project was created while learning fundamental programming concepts in C#.  
The goal is to practise syntax, understand core programming logic, and build a strong foundation for more advanced software development.
